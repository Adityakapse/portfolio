Amazon clone

Smartbrain

Created a web application that leverages machine learning to detect faces in images. The front end, built with React.js and enhanced with particle.js, offers a visually engaging and interactive user interface. The back end, powered by Node.js and Express.js, handles server-side logic and communication with a relational PLSQL database. The application features user registration and authentication, allowing users to securely access their accounts. Additionally, a unique counting system was implemented to track how many times each user detects faces, providing a personalized experience. The entire application was developed with a clear separation of concerns, deploying both the front end and back end independently on Heroku for scalability and ease of maintenance.

Attendance System using Face Recognition

Developed an automated attendance system that utilizes face recognition technology to streamline the attendance process. Built with Python and powered by OpenCV, the system captures real-time video input, accurately detects and extracts faces, and then matches them against a pre-recorded database of authorized individuals. The system ensures efficient and reliable attendance tracking by automating the identification process, minimizing manual errors, and enhancing security. This project demonstrates advanced use of computer vision techniques and showcases the potential for face recognition in everyday applications.
